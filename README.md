# Chadneal Gimage

## How do I install these formulae?

`brew install chadneal/gimage/<formula>`

Or `brew tap chadneal/gimage` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "chadneal/gimage"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
